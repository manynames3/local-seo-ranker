# Demo Script

## One-Minute Version

Local SEO Ranker is a territory visibility console for local SEO agencies. It turns a business, keyword, city, and competitors into a geo-grid ranking view and a build plan.

Today the UI can run a transparent demo estimate. The production path is already shaped around Scrappa: when the Cloudflare backend has `SCRAPPA_API_KEY` and center coordinates, `/api/scans` can run coordinate-based Google Maps checks server-side and feed live rank cells into the same report.

## Walkthrough

1. Open `/ ?demo=1` or click **Load sample report**.
2. Point out the top status: the report is explicit about whether it is an estimate or live Scrappa evidence.
3. Show the local ranking view first. Explain that each cell represents one point in the market grid.
4. Move to Market pulse and Strategic read. These summarize the ranking weakness and first operational move.
5. Scroll to Topic gaps, Entity signals, Internal links, and AI search readiness. These convert visibility gaps into execution tasks.
6. Use JSON, CSV, Print, or Copy to show that the report can become an agency deliverable.

## Buyer Positioning

"This helps an agency show a local business exactly where they are weak in the city and what to build next. The current demo shows the strategy layer. The live scan path connects Scrappa so the map is based on real Maps/SERP evidence instead of estimates."

## Do Not Claim Yet

- Do not claim the current static demo is live Google Maps rank tracking.
- Do not claim billing, accounts, saved reports, or scheduled scans exist.
- Do not sell recurring rank tracking until scan persistence and live provider QA are complete.
